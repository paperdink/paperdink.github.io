1. Getting Started

Installing dependencies

Install Arduino IDE#

Please download the latest compatible version of Arduino IDE from Arduino’s site →

Install ESP32 Board in Arduino IDE#

  1. In the Arduino IDE, go to File > Preferences

  2. Find the Additional Board Manager URLs field to enter the following URL and click OK

https://dl.espressif.com/dl/package_esp32_index.json
  1. Go to Tools > Board > Boards Manager & install ESP32 by Espressif Systems board by searching for the same.
  2. You should be able to see ESP32 Arduino boards in Tools > Board

Install ESP32 Sketch Data Upload#

  1. Download the esptool from here →
  2. Go to the directory Arduino > Tools folder. For Windows it looks like this:
C:\Program Files\Arduino\tools
  1. Unzip the downloaded ESP32FS-1.0.zip or Source code file here and you should have the following file structure:
<home_dir>/Arduino/tools/ESP32FS/tool/esp32fs.jar
  1. Restart Arduino IDE. You should see ESP32 Sketch Data Upload under the Tools menu. Make sure you have selected ESP32 Dev Module Board in Tools > Board > ESP32 Arduino > ESP32 Dev Module.
