1. Getting Started

Installing dependencies

Install Arduino IDE#

Please download the latest compatible version of Arduino IDE from Arduino’s site →

Install ESP32 Board in Arduino IDE#

  1. In the Arduino IDE, go to File > Preferences

  2. Find the Additional Board Manager URLs field to enter the following URL and click OK

https://dl.espressif.com/dl/package_esp32_index.json
  1. Go to Tools > Board > Boards Manager & install ESP32 by Espressif Systems board by searching for the same.
  2. You should be able to see ESP32 Arduino boards in Tools > Board
  3. paperd.ink works with the ESP32 Dev Module board.
